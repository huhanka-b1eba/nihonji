# Nihonji — современный аниме-каталог на React

**Nihonji** — функциональный веб-сервис для просмотра и поиска аниме, построенный на базе публичного REST API Jikan.
Проект ориентирован на высокую производительность, удобство пользователя, масштабируемость и чистую архитектуру по методологии **Feature-Sliced Design**.

Приложение реализовано на продакшен-уровне с использованием современных frontend-практик и продуманным пользовательским опытом.

---

## Основные возможности

### Просмотр аниме

* Топы аниме по категориям
* Детальные страницы тайтлов
* Персонажи, студии, трейлеры
* Похожие тайтлы и рекомендации
* Поиск по названию с debounce
* Фильтрация и сортировка

### Пользовательские функции

* Избранное
* История просмотров
* Лайки
* Сохранение данных в `localStorage`
* Восстановление позиции скролла

### Интерфейс

* Светлая и тёмная темы
* Анимации (Framer Motion)
* Skeleton-загрузки
* Error Boundary для обработки ошибок
* Пагинация и infinite scroll
* Адаптивная вёрстка
* UI-kit на SCSS и CSS Modules

---

## Архитектура

Проект построен по методологии **Feature-Sliced Design**:

```text
src/
  app/          провайдеры, глобальные стили, роутинг, стор
  pages/        страницы приложения
  widgets/      крупные интерфейсные блоки
  features/     пользовательская логика
  entities/     доменные сущности (Anime, Character, Studio)
  shared/       UI-kit, утилиты, API, типы
```

### Преимущества подхода

* Масштабируемость проекта
* Чёткое разделение ответственности
* Простота поддержки
* Готовность к росту функциональности

## API

Используется открытый REST API **Jikan**.

Реализовано:

* Полная типизация ответов
* RTK Query endpoints
* Кеширование запросов
* Автоматический refetch
* Частичная нормализация данных
