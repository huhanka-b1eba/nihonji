Nihonji — современный аниме-каталог на React

Nihonji — это функциональный веб-сервис для просмотра и поиска аниме, построенный на базе публичного REST API Jikan.
Проект ориентирован на высокую производительность, удобство пользователя, масштабируемость и чистую архитектуру по методологии Feature-Sliced Design.

Nihonji представляет собой продакшен-уровень frontend-приложение с современным технологическим стеком и продуманным UX.

Архитектура

Проект реализован по методологии Feature-Sliced Design:

src/
  app/          провайдеры, глобальные стили, роутинг, стор
  pages/        страницы приложения
  widgets/      крупные интерфейсные блоки
  features/     пользовательская логика
  entities/     доменные сущности (Anime, Character, Studio)
  shared/       UI-kit, утилиты, API, типы

Такой подход обеспечивает:

масштабируемость проекта

изоляцию бизнес-логики

удобство поддержки и развития

соответствие продакшен-стандартам
